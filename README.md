# ☕ Coffee Shop Sales Analysis Dashboard – Power BI Project

## 📌 Project Overview
This project showcases an interactive Power BI dashboard developed for analyzing coffee shop sales performance. The report provides meaningful business insights related to sales trends, product performance, customer purchasing behavior, and store operations.

The dashboard is designed to help business owners and analysts make data-driven decisions by visualizing key performance indicators in a simple and interactive way.

---

## 🎯 Objectives
- Analyze overall coffee shop sales performance
- Track daily and monthly sales trends 
- Identify top-selling products and categories
- Understand customer purchasing patterns
- Compare Weekday and Weakend sales
- Evaluate store location performance 
---

## 📊 Dashboard Details

 📌 This interactive dashboard includes the following insights:

- 📅 Yearly Sales Analysis
- 📈 Monthly Sales Trends
- 🗓️ Quarterly Sales Performance
- ☕ Product Category wise Sales
- 🔝 Top 10 Products by Sales
- 🏪 Sales by Store Location
- 📅 Daily Sales Insights
- 📊 Weekday vs Weekend Sales Comparison

The dashboard also contains interactive slicers and filters for better data exploration and analysis.
